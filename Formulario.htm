Para subir tu archivo Formulario Pedido.html a GitHub, debes seguir estos pasos. Aquí te explico cómo hacerlo manualmente y también cómo preparar un repositorio básico.

✅ OPCIÓN 1: Subida Manual vía GitHub

  1. Ve a https://github.com y entra con tu cuenta.

  2. Crea un nuevo repositorio: New > Repository.

  3. Ponle un nombre (ej. formulario-pedidos), selecciona público o privado y haz clic en Create repository.

  4. En la nueva página del repositorio, haz clic en "Add file" > "Upload files".

  5. Arrastra tu archivo Formulario Pedido.html o haz clic para seleccionarlo.

  6. Haz clic en el botón verde "Commit changes".

✅ OPCIÓN 2: Crear estructura de proyecto con Git

Si quieres subirlo desde tu computador con Git, sigue estos pasos:

1. Crea una carpeta y mueve el archivo:

bash
mkdir formulario-pedidos
cd formulario-pedidos
mv /ruta/a/Formulario\ Pedido.html index.html

2. Inicia un repositorio Git:

bash
git init
git add index.html
git commit -m "Formulario de pedidos inicial"

3. Conecta con un repositorio en GitHub:

Primero crea el repositorio en GitHub, luego en tu terminal:

bash
git remote add origin https://github.com/tu-usuario/formulario-pedidos.git
git branch -M main
git push -u origin main

Reemplaza tu-usuario por tu nombre de usuario real en GitHub.

🧪 Verlo Online

Puedes ver el formulario directamente online con GitHub Pages:

  1. Ve a tu repositorio en GitHub.

  2. Entra a Settings > Pages.

  3. Bajo "Source", selecciona main y carpeta raíz / (root).

  4. Guarda y espera unos segundos.

  5. Visita la URL pública que GitHub te muestra (algo como https://tu-usuario.github.io/formulario-pedidos).

# Formulario de Pedidos - Uniformes Deportivos

Este proyecto es un formulario interactivo para registrar pedidos de uniformes deportivos, incluyendo nombres de jugadores, tallas, tipo de prenda y otras opciones.

## 🚀 Características

- Añadir jugadores dinámicamente
- Contador de jugadores y petos
- Exportación automática a Excel
- Vista previa para compartir en WhatsApp
- Interfaz moderna con TailwindCSS
- Modal animado para compartir

## 📦 Tecnologías usadas

- HTML5
- TailwindCSS
- Font Awesome
- XLSX.js (para generar Excel)
- JavaScript puro (sin frameworks)

## 📄 Vista previa en GitHub Pages
